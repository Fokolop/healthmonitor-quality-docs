# healthmonitor-quality-docs
Сервис HealthMonitor предназначен для мониторинга состояния здоровья распределенных систем и приложений. Он предоставляет REST API для сбора метрик, настройки проверок и оповещений о критических состояниях. Система разработана с учетом требований высокой доступности и масштабируемости.
